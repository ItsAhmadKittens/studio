# **App Name**: Figma Translator

## Core Features:

- Frame Selection and Text Extraction: Allows users to select multiple frames in Figma and extracts all text content from them.
- Language Detection: Detects the default language of the extracted text.
- Translation Selection: Provides a scrollable list of available translation languages for the user to choose the target language.
- Text Translation: Translates each text element within the selected frames to the chosen target language using an AI translation tool.
- Content Replacement: Replaces the original text in the Figma frames with the translated text.
- Translation Status Notification: Provides visual feedback to indicate the status of the translation process (e.g., success, failure).

## Style Guidelines:

- Primary color: Desaturated purple (#9A82DB) for a modern and calming feel, reflecting design collaboration.
- Background color: Light gray (#F4F4F8), providing a neutral backdrop for the plugin interface.
- Accent color: Bright magenta (#E91E63) for interactive elements and highlights to draw attention to key actions.
- Body and headline font: 'Inter' for a clean, modern, and highly readable interface.
- Use clear and simple icons to represent actions like selecting frames, choosing languages, and initiating translation.
- Maintain a clean and organized layout, grouping related functions together to minimize cognitive load for users.
- Use subtle animations and transitions to provide feedback and enhance the user experience, such as progress indicators during translation.
